# Demo Results

Pre-generated artefacts from a real SynapticFour Showcase run.

Included files:

- `benchmark.json` — precision/recall/F1 benchmark output
- `metrics.json` — run metrics including WES run identifiers
- `drs_micro.json` — DRS timing/micro benchmark snapshot
- `helios-report.json` — HELIOS audit export
- `showcase-report.json` — merged machine-readable summary
- `showcase-report.md` — stakeholder-friendly summary
- `drs-object-link.txt` — example DRS object identifier and access endpoint
